Tracking Your Itinerary

To use the Coverage Tracker from the main menu you must have GPS enabled. It works best if you also have NETWORK access through WiFi or Mobile.

Pressing Start will create a new Expedition, sending your points to the POSIT server while storing them on the phone. After you Stop tracking, you can save your expedition. Use the List button to view previously saved tracks.

If network connectivity is lost, Tracker will cache the points on the phone for later transmission. Known issue: The responsiveness of the user interface degrades when too many points are cached. We're working on it.
